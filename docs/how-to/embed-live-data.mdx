# Embed live data

Docs can embed live query results with no code change — just authored markup.
Add a `data-sql-table` container with an inline SQL query. The client island
runs it through the guarded `/api/query.json` endpoint and renders a table.

When the query engine is not configured, the embed quietly reports
"Live data unavailable" and the "Show SQL" disclosure still works — the page
never errors.

## Example

<div data-sql-table data-title="Recent orders">
  <script type="text/plain" data-sql>
    SELECT id, customer, total
    FROM orders
    ORDER BY created_at DESC
    LIMIT 10
  </script>
</div>

## Rules

- Queries must be a single `SELECT` (or `WITH … SELECT`) statement.
- Results are row-capped and time-capped server-side.
- This page is `.mdx` because embeds need raw HTML — see the
  [authoring formats reference](../reference/authoring-formats.md).
